# 🚀 Tarea: Colecciones y Streams en Java

¡Hola! En este repositorio presento la resolución de la guía práctica sobre **Colecciones** y **Streams** en Java. El objetivo de este proyecto es demostrar el uso correcto de diferentes estructuras de datos dinámicas y el procesamiento moderno de elementos.

---

## 🛠️ Ejercicios Desarrollados

### 📂 1. Registro de Productos (`ArrayList`)
* **Carpeta:** `EJERCICIO1-ARRAYLIST`
* **Concepto:** Uso de listas ordenadas y dinámicas. El programa solicita al usuario una cantidad de productos, los almacena secuencialmente y luego los despliega en una lista limpia junto al total de elementos ingresados usando `.size()`.

### 📂 2. Registro de Usuarios Únicos (`HashSet`)
* **Carpeta:** `EJERCICIO2-HASHSET`
* **Concepto:** Almacenamiento sin duplicados. Se implementó un sistema que impide el registro de nombres de usuario repetidos utilizando la propiedad booleana del método `.add()`. Si el usuario ya existe, el sistema avisa con un mensaje en consola.

### 📂 3. Agenda Telefónica (`HashMap`)
* **Carpeta:** `EJERCICIO3-MAP`
* **Concepto:** Relaciones Clave-Valor (`Key-Value`). Se creó una agenda que almacena pares de datos (Nombre y Teléfono). Incluye el recorrido del mapa mediante `entrySet()` con `.getKey()` y `.getValue()`, además de una función de búsqueda directa por nombre usando `.get()`.

### 📂 4. Filtrar Notas Aprobadas (`Stream`)
* **Carpeta:** `EJERCICIO4-STREAM`
* **Concepto:** Procesamiento de datos con Java Streams y expresiones lambda. El programa toma una lista de notas decimales (`Double`), las filtra para mostrar solo aquellas mayores o iguales a `4.0` usando `.filter()`, y cuenta el total de aprobados de forma eficiente con `.count()`.
